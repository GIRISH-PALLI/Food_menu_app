# Food Menu Website

A modern, interactive food menu website showcasing vegetarian and non-vegetarian cuisine items with an elegant image gallery.

---

## Features

- **Dual Menu System** - Separate pages for vegetarian and non-vegetarian items
- **Image Gallery** - Browse food items through an organized photo collection
- **Responsive Design** - Clean, professional styling with CSS
- **Interactive Elements** - JavaScript-powered user experience
- **Well-Organized Structure** - Easy navigation across all pages

---

## Project Structure

### Core Pages
| File | Purpose |
|------|---------|
| `index.html` | Main landing page |
| `veg.html` | Vegetarian items menu |
| `non-veg.html` | Non-vegetarian items menu |

### Menu Item Pages
- **Vegetarian**: `veg1.html` - `veg4.html`
- **Non-Vegetarian**: `nv1.html` - `nv5.html`
- **Additional Pages**: `page1.html` - `page4.html`

### Media & Gallery
- `img.html` - Main gallery hub
- `img1.html` - `img13.html` - Individual image showcase pages

### Styling & Scripts
- `style.css` - Primary stylesheet
- `ind.css` - Additional styling
- `script.js` - Interactive functionality

### Supporting Files
- `man.xml` - Data file
- `m.java` - Java utility
- `cs.html` - Miscellaneous page

---

## Quick Start

1. Open `index.html` in your web browser
2. Navigate through the vegetarian and non-vegetarian menus
3. Explore the image gallery
4. Enjoy the interactive experience!

---

## Technologies Used

- **HTML5** - Document structure
- **CSS3** - Styling and layout
- **JavaScript** - Interactivity and functionality
